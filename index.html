HTML

  • Handles content and structure of a website
  • The head element contains information, or metadata, that helps render the page correctly
  • The body element represents the visible content shown to the user
  • Section Elements define a section in a document that contains relevant thematic content

CSS

  • Programming language used by developers to define how a website's content is displayed to the user.
  • Defines the colors, aesthetics, visual layout, and organization.
  • Margin property indicates how much space we want outside of an element
  • Padding property indicates how much space we want around the content that is inside an element

GIT

  • Version control system that lets you manage and keep track of each version of your project
  • Command always starts with the word git, which differs from bash commands
  • git status-checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git checout main: sends you back to main branch
  • git pull origin main: pulls down the latest main branch

JAVASCRIPT

  • High-level dynamic programming language that allows for interactivity with the website
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • An array is a single variable that is used to hold a group of data
  • (for) loops follows a predictable pattern to perform a task repeatedly
